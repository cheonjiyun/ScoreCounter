# 점수 카운터 ScoreCounter

자격증 등 공부를 하면서 모의고사 점수 계산이 필요할 때 도구로 쓸 수 있습니다.

1. 시험 종류에 맞추어 문제 개수를 설정합니다.

2. 맞힌 문제 개수를 셉니다.

3. 100점 단위인 환산 점수를 얻을 수 있습니다
